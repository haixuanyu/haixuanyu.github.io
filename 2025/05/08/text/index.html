我的第一篇博客

这是我第一篇博客的文章，将会讲述自己这篇博客是如何搭建的。

1. 用到的工具

  • 前端 Node.js
  • githubpage GitHub 提供的一个免费的静态网站托管服务GitHub
  • 博客框架Hexo
  • Markdown一种标记语言，这篇文章就是使用markdown写的

2. 搭建博客

  1. 创建一个github账号以及一个github仓库
  2. 安装 Node.js
  3. 安装 Git
  4. 通过命令行运行 npm install -g hexo-cli 安装 Hexo
  5. 部署到githubpage
  6. 使用hexo主题，例如我使用的是butterfly主题
  7. 配置yml，进行一定程度的自定义
  8. 通过markdown语言写这篇文章

3.结语

有些人看博客不喜欢说话，但会反馈bug和美术设计问题，这样的人帅气低调有内涵，美丽大方很优雅[doge]
